# Twitchy
### A Twitch inspired theme for OBS v28+ to fit the Twitch integration docks

How to install the version with the custom font:

1. Download the Twitchy.zip [here](https://github.com/derMiepz/Twitchy/releases/tag/Release)
2. Extract Twitchy.zip
3. Open Twitchy (with Font) -> FONT
4. Install the font in there
5. Move the **Twitchy** folder and **Yami_Twitchy.ovt** file into your theme folder (Default is: ``C:\Program Files\obs-studio\data\obs-studio\themes\``)
6. Restart OBS in case it's already open
7. In OBS, go to Settings -> Appearance -> Style and select the theme
8. Click OK

Enjoy your new theme!
